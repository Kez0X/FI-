
# 📐 Fonctions Quadratiques

Les **fonctions quadratiques** sont très importantes en mathématiques, car elles décrivent des phénomènes qui suivent une forme parabolique. Découvrons ensemble ces fonctions fascinantes ! 😎



## 🔍 Qu'est-ce qu'une Fonction Quadratique ?

Une fonction quadratique est une fonction de la forme :

$$
f(x) = ax^2 + bx + c
$$

où :
- $$a$$, $$b$$ et $$c$$ sont des **coefficients**,
- $$a$$ ne peut pas être égal à zéro, sinon ce ne serait plus une fonction quadratique mais une fonction affine.

### Exemple :

Si on a la fonction :

$$
f(x) = 2x^2 - 3x + 1
$$

Cela signifie que la fonction dépend de $$x$$ de manière quadratique. Par exemple :
- Pour $$x = 0$$, $$f(0) = 2(0)^2 - 3(0) + 1 = 1$$.
- Pour $$x = 1$$, $$f(1) = 2(1)^2 - 3(1) + 1 = 2 - 3 + 1 = 0$$.



## 📚 Propriétés des Fonctions Quadratiques

Les fonctions quadratiques ont des propriétés intéressantes, particulièrement à cause de leur forme parabolique.

### 1. **Forme Parabolique**

Le graphique d'une fonction quadratique est une **parabole**. La forme de la parabole dépend du signe de $$a$$ :
- Si $$a > 0$$, la parabole est **ouverte vers le haut** (comme un bol 🍲),
- Si $$a < 0$$, la parabole est **ouverte vers le bas** (comme un chapeau 🎩).

### 2. **Sommet de la Parabole**

Le sommet de la parabole est un point très important. Il représente soit le point le plus bas (si $$a > 0$$) ou le point le plus haut (si $$a < 0$$).

Le sommet de la parabole pour la fonction $$f(x) = ax^2 + bx + c$$ se trouve en $$x = \frac{-b}{2a}$$. Une fois que tu as trouvé $$x$$, tu peux calculer $$f(x)$$ pour trouver l'ordonnée du sommet.

### Exemple :

Pour la fonction $$f(x) = 2x^2 - 3x + 1$$, on peut trouver le sommet en utilisant $$x = \frac{-(-3)}{2(2)} = \frac{3}{4}$$. Puis, on remplace dans la fonction pour trouver l'ordonnée :

$$
f\left(\frac{3}{4}\right) = 2\left(\frac{3}{4}\right)^2 - 3\left(\frac{3}{4}\right) + 1 = 2 \times \frac{9}{16} - \frac{9}{4} + 1 = \frac{18}{16} - \frac{36}{16} + \frac{16}{16} = \frac{-2}{16} = -\frac{1}{8}
$$

Donc, le sommet de cette parabole est le point $$\left(\frac{3}{4}, -\frac{1}{8}\right)$$.



## 📈 Graphique d'une Fonction Quadratique

Le graphique d'une fonction quadratique est une parabole. Voici comment tracer la parabole d'une fonction quadratique :

1. Trouve le sommet de la parabole en utilisant la formule $$x = \frac{-b}{2a}$$.
2. Remplace $$x$$ dans la fonction pour obtenir l'ordonnée du sommet.
3. Trouve quelques points supplémentaires en choisissant des valeurs de $$x$$ et en calculant les valeurs correspondantes de $$f(x)$$.
4. Trace la parabole qui passe par ces points.

### Exemple de graphique :

Pour la fonction $$f(x) = 2x^2 - 3x + 1$$, nous avons trouvé que le sommet est $$\left(\frac{3}{4}, -\frac{1}{8}\right)$$. Maintenant, nous allons calculer quelques autres points :
- Pour $$x = 0$$, $$f(0) = 1$$, donc le point est $$P(0, 1)$$.
- Pour $$x = 1$$, $$f(1) = 0$$, donc le point est $$P(1, 0)$$.

En traçant ces points et en rejoignant avec une courbe douce, on obtient une parabole.

```
      |
    1 | .  
      |   .
    0 |     .
      |        .
   -1 |          .
      |______________
            0   1   2
```



## 📍 Résolution des Équations Quadratiques

Une équation quadratique est une équation de la forme :

$$
ax^2 + bx + c = 0
$$

Il existe plusieurs méthodes pour résoudre ces équations, telles que :
- La **méthode du discriminant**,
- La **méthode du produit nul** (si l'équation peut être factorisée),
- La **formule quadratique**.

La **formule quadratique** est donnée par :

$$
x = \frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$

Le discriminant, $$\Delta = b^2 - 4ac$$, permet de déterminer le nombre de solutions de l'équation :
- Si $$\Delta > 0$$, il y a deux solutions réelles distinctes,
- Si $$\Delta = 0$$, il y a une solution réelle double,
- Si $$\Delta < 0$$, il n'y a pas de solution réelle.



## 📝 Applications des Fonctions Quadratiques

Les fonctions quadratiques sont utilisées pour modéliser des phénomènes où il y a une relation de type carré entre les variables. Par exemple :
- Le mouvement d'un objet en chute libre (lorsque la résistance de l'air est négligeable).
- Les trajectoires des projectiles.
- Les calculs de surfaces et volumes dans certaines géométries.



## 🧮 Exemple d'Équation Quadratique à Résoudre

Résolvons l'équation quadratique suivante :

$$
2x^2 - 3x - 5 = 0
$$

1. Identifions les coefficients : $$a = 2$$, $$b = -3$$, et $$c = -5$$.
2. Calculons le discriminant :

$$
\Delta = (-3)^2 - 4(2)(-5) = 9 + 40 = 49
$$

3. Appliquons la formule quadratique :

$$
x = \frac{-(-3) \pm \sqrt{49}}{2(2)} = \frac{3 \pm 7}{4}
$$

4. Les deux solutions sont :

$$
x_1 = \frac{3 + 7}{4} = \frac{10}{4} = 2,5 \quad \text{et} \quad x_2 = \frac{3 - 7}{4} = \frac{-4}{4} = -1
$$

Les solutions sont donc $$x = 2,5$$ et $$x = -1$$.



## 📝 Conclusion

Les **fonctions quadratiques** sont essentielles pour comprendre les phénomènes qui suivent une forme parabolique. Elles sont modélisées par l'expression $$f(x) = ax^2 + bx + c$$ et ont un graphique en forme de parabole. Grâce à la formule quadratique, nous pouvons résoudre des équations quadratiques et trouver les solutions d'un problème.



Et voilà, tu maîtrises désormais les **fonctions quadratiques** ! 🎉
